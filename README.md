# VesystemRCE
和信创天云桌面 远程命令执行漏洞 

# Options

optional arguments:

  -h, --help            show this help message and exit
  
  -u url, --url url     Target url eg:"http://127.0.0.1"
  
  -f file, --file file  Targets in file eg:"ip.txt"
  
# Use

python3 VesystemRCE.py -u http://127.0.0.1

python3 VesystemRCE.py -f ip.txt
